Procedure

Step 1: Select a language which you know better

Step 2: Select the buttons which has words written on it, in a proper order

Output: Group of words in a selected order will be shown

Note:

  1. If a wrong sentence is formed, Re-form the sentence button is available for Re-setting.
  2. You can check whether the formed sentence is a valid or not by clicking Check the correctness of the sentence button.
  3. For a wrong sentence, you can get the correct sentence by clicking Get correct sentence button.
